# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

Design Decisions

The design and development of this website were guided by principles of usability, responsiveness, and maintainability.

To ensure a responsive and consistent layout across multiple devices, the Bootstrap framework was utilized. This enabled rapid development through the use of pre-designed templates, grid systems, and UI components.

User experience was further enhanced by incorporating efficient visual effects and transitions. These effects were carefully implemented to maintain a balance between interactivity and performance.

Typography and color schemes were strategically selected to establish a clear visual hierarchy and improve readability. The use of varied font styles and coordinated colors contributes to an aesthetically pleasing and engaging interface.

In addition, images were transformed into sticker-style graphics to introduce a creative and distinctive visual element. This approach enhances the overall look and feel of the website.

Emojis were also integrated where appropriate to create a more interactive and expressive user interface, aligning with modern web design trends.

The application was developed using React, with a strong emphasis on reusable components. This component-based architecture improves code reusability, scalability, and ease of maintenance.

Furthermore, custom CSS styles were applied alongside framework-provided styles to achieve specific design requirements and provide greater control over the visual presentation.

Overall, these design decisions result in a modern, responsive, and user-centered web application.

I also used custom CSS to design specific parts of the website and improve the overall appearance.

These decisions helped me create a modern and user-friendly website.
